접속을 차단하고 싶은 사이트의 URL에 포함될 키워드를 한 줄에 하나씩 입력하세요. (예: example.com/board)

드래그 기능 비활성화

Shift/Ctrl/Alt + 드래그 기능을 비활성화 하고 싶은 사이트의 호스트 주소를 한 줄에 하나씩 입력하세요. (예: example.com)

💾

데이터 관리

모든 설정을 하나의 파일로 백업하거나, 백업 파일로부터 설정을 복원합니다.

📑

1. 브라우저 및 탭 관리 (전역 기능)

브라우저 전체에 적용되는 탭 관리 및 패널 기능입니다.

기능 단축키 / 동작 설명
사이드패널 열기 Alt + L LunaTools 사이드패널(여러 URL 열기, 세션 매니저)을 엽니다.
탭 정렬 Alt + A 현재 창의 탭들을 도메인 및 URL 경로 순서대로 정렬합니다.
탭 합치기 아이콘 우클릭 확장 프로그램 아이콘 우클릭 메뉴 > '모든 탭을 하나의 창으로 합치기'
🧭

2. 웹 페이지 탐색 및 링크 (웹페이지 내 동작)

웹 페이지 내부에서 빠르게 이동하거나 링크를 처리하는 기능입니다.

기능 단축키 / 동작 설명
이전 페이지 ← (방향키) URL의 숫자를 인식하여 이전 페이지로 이동합니다.
다음 페이지 → (방향키) URL의 숫자를 인식하여 다음 페이지로 이동합니다.
상위 경로 이동 Alt + ↑ 현재 URL의 상위 경로 또는 상위 도메인으로 이동합니다.
최상위 경로 이동 Alt + Ctrl + ↑ 현재 사이트의 최상위 도메인(Root)으로 즉시 이동합니다.
링크 열기 Shift + 드래그 드래그 영역 내의 링크들을 새 탭으로 즉시 엽니다.
링크 복사 Ctrl + 드래그 드래그 영역 내의 링크 주소들을 클립보드에 복사합니다.
지연 열기 Alt + 드래그 드래그 영역 내의 링크들을 2초 간격으로 순차적으로 엽니다.
마우스 제스처 우클릭 + 드래그 ↑: 새로고침, ↓: 탭 닫기, ←: 뒤로 가기, →: 앞으로 가기
🎬

3. 미디어 제어 (영상/음성)

유튜브, 동영상 사이트 등에서 미디어를 편리하게 제어합니다.

기능 단축키 / 동작 설명
현재 탭 음소거 Alt + M 현재 탭의 소리를 끄거나 켭니다 (토글).
전체 창 음소거 Shift + Alt + M 모든 창의 소리를 끄거나 켭니다 (토글).
PiP 모드 Ctrl + Shift + P 현재 재생 중인 동영상을 Picture-in-Picture(작은 창) 모드로 전환합니다.
볼륨 부스터 Alt + V 현재 탭의 소리 크기를 3배(300%)로 증폭합니다. (토글 방식)
영상 회전 Ctrl + Shift + Alt + R 동영상을 90도씩 회전시킵니다.
⚡

4. 유틸리티 (스마트 변환)

환율, 길이·넓이·평수, 저장 용량, 데이터 속도, 이동 속도, 압력, 시간 등을 드래그하여 즉시 변환합니다.

기능 단축키 / 동작 설명
스마트 변환 Alt + Z 웹페이지에서 텍스트(환율, 길이·넓이·평수, 저장 용량, 데이터 속도, 이동 속도, 압력, 시간 등)를 드래그한 후 누르면 변환 결과를 팝업으로 보여줍니다.
팝업 닫기 Esc 스마트 변환 결과 팝업 창을 닫습니다.

LunaTools Alt+Z Cheat Sheet

웹페이지에서 바꾸고 싶은 텍스트를 선택한 뒤 Alt + Z. 이것만 기억하시면 됩니다.

10초 사용법

순서 이렇게 하세요 알아두면 편한 점
1 웹페이지에서 변환할 글자를 선택합니다. 문장에 여러 단위가 있으면 함께 찾아줍니다.
2 Alt + Z를 누릅니다. Ctrl·Shift는 함께 누르지 않습니다.
3 선택 영역 옆 팝업에서 결과를 확인합니다. 각 결과의 복사 버튼으로 바로 복사할 수 있습니다.
4 Esc 또는 ×로 닫습니다. 제목 표시줄은 드래그할 수 있고, 페이지를 스크롤해도 닫힙니다.

엔·위안 기호 ¥·￥는 일본 엔으로 처리합니다. 중국 위안은 CNY 또는 위안을 씁니다.
